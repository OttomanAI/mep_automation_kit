DWG FILES
=========
 
This folder is for Civil 3D drawing (.dwg) files containing the pipe networks.
 
These are the source files used to export the LandXML (.xml) files.
Place your .dwg files here for reference and version control.
 
To export LandXML from Civil 3D:
1. Open the .dwg file in Civil 3D
2. Select the pipe network(s) to export
3. Go to Output > Export to LandXML
4. Save the .xml file into 01_XML_Data/
5. Run the parser (run.bat) from the drainage_processor/ root folder
 
Example .dwg files are provided in example_dwg_Files/ for reference.
